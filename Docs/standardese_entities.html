• pants::fs

    • FileType - Type of a file

    • Metafile - Describes a file

    • Superblock - The master block for the file system

    • UnusedMetafile - When a metafile is unused, it participates in a linked list of unused metafiles

    • g_block_size - Size of each file system block in bytes

    • g_blocks_per_block_group - Number of blocks in each block group

    • g_blocks_per_metafile - Number of file system blocks per metafile

    • g_first_block_group_block_id - The block ID of the first block group

    • g_invalid_block_id - Used to mark an invalid block ID

    • g_magic_number - Magic number identifying a PantsFS file system

    • g_metafile_size - Size of each metafile in bytes

    • g_metafiles_per_block_group - Number of metafiles per block group
